Инструкция по работе с Git
1. Установка Git:
Скачайте Git с официального сайта.
Установите Git, следуя инструкциям для вашей операционной системы.
2. Создание репозитория:
Инициируйте новый репозиторий в вашем проекте:
git init
3. Добавление файлов в репозиторий:
Добавьте все файлы в репозиторий:
git add .
4. Создание коммита:
Создайте коммит с описанием изменений:
git commit -m "Описание изменений"
5. Работа с ветками:
Создайте новую ветку:
git branch название_ветки
Переключитесь на созданную ветку:
git checkout название_ветки
6. Получение обновлений из удаленного репозитория:
Добавьте удаленный репозиторий (например, GitHub):
git remote add origin https://github.com/ваш_аккаунт/ваш_репозиторий.git
Получите изменения:
git pull origin основная_ветка
7. Отправка изменений в удаленный репозиторий:
Отправьте изменения на сервер:
git push origin ваша_ветка
8. Работа с подмодулями (если применимо):
Добавьте подмодуль:
git submodule add https://github.com/путь_к_подмодулю.git
Обновите подмодули:
git submodule update --init --recursive
Пример архивирования проекта:
Создайте архив с проектом:
tar -czvf ваш_проект.tar.gz ваш_проект
Скачайте архив с проектом для дальнейшей работы.
Важно!
Перед отправкой изменений в репозиторий, удостоверьтесь, что вы работаете в правильной ветке.
Документируйте ваши коммиты подробно, чтобы облегчить понимание изменений другим разработчикам.
Это краткая инструкция по использованию Git. Успешной работы над вашим проектом!
